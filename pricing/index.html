🔧 管理员：支付集成指南

1. 面包多 (mianbaoduo.com) 设置

注册面包多账号 → 创建商品 → 设置自动发货 → 获取购买链接

1. 访问 https://mianbaoduo.com 注册账号 2. 进入「卖家中心」→「创建商品」 3. 填写商品名称、价格、描述 4. 选择「自动发货」→ 上传卡密或设置兑换码 5. 获取商品链接，替换页面中的 https://mianbaoduo.com 6. Webhook回调地址：https://nextool.dev/api/payment/callback

2. LemonSqueezy 设置

注册LemonSqueezy → 创建产品 → 配置Webhook → 获取Checkout链接

1. 访问 https://www.lemonsqueezy.com 注册账号 2. 进入 Dashboard → Create Product 3. 设置价格（支持多币种） 4. 配置 Webhook：Settings → Webhooks 5. Webhook URL: https://nextool.dev/api/payment/lemonsqueezy 6. 事件选择：subscription_created, subscription_cancelled, payment_success 7. 获取 Checkout 链接，替换页面中的链接

3. 爱发电 (afdian.com) 设置

注册爱发电 → 创建发电计划 → 获取赞助链接

1. 访问 https://afdian.com 注册创作者账号 2. 进入「创作者中心」→「发电计划」 3. 创建月度发电计划，设置对应价格 4. 获取赞助链接，替换页面中的 https://afdian.com 5. Webhook设置：创作者中心 → 开发者 → Webhook 6. 回调地址：https://nextool.dev/api/payment/afdian

4. 连接支付链接
