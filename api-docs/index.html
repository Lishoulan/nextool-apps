📖 概述

ViralCopy API 帮助你快速生成适合各大社交平台的爆款文案。只需提供关键词和目标平台，即可获得多组高质量文案。

Base URL: https://api.nextool.app

响应格式: JSON  |  编码: UTF-8  |  协议: HTTPS

📌 生成文案

POST /api/v1/generate 根据关键词生成爆款文案

请求参数

参数类型必填说明
keywordstring必填文案关键词，如 "健身餐"
platformstring必填目标平台: xiaohongshu / douyin / weibo / wechat / bilibili / zhihu
tonestring可选语气: casual / professional / humorous / inspiring（默认: casual）
countinteger可选生成数量 1-5（默认: 3）
languagestring可选语言: zh / en（默认: zh）

请求示例
