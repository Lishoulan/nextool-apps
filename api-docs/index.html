接口文档

POST /api/chat AI文本生成接口

核心接口，支持各类AI文本生成任务。通过 messages 数组传入对话上下文，模型将返回智能生成的文本内容。

请求头

参数名类型必填说明
Content-Type string 必填 application/json

请求体参数

参数名类型必填说明
messages array 必填 对话消息数组，每项包含 role 和 content
messages[].role string 必填 角色：system / user / assistant
messages[].content string 必填 消息内容
model string 可选 模型名称，默认 deepseek-chat

代码示例

curl -X POST https://nextool-api-proxy-vercel.vercel.app/api/chat \
  -H "Content-Type: application/json" \
  -H "X-RapidAPI-Key: your-rapidapi-key" \
  -H "X-RapidAPI-Host: nextool-api-proxy-vercel.vercel.app" \
  -d '{
    "messages": [
      {"role": "user", "content": "请帮我写一段关于人工智能的摘要"}
    ],
    "model": "deepseek-chat"
  }'
import requests

url = "https://nextool-api-proxy-vercel.vercel.app/api/chat"
headers = {
    "Content-Type": "application/json",
    "X-RapidAPI-Key": "your-rapidapi-key",
    "X-RapidAPI-Host": "nextool-api-proxy-vercel.vercel.app"
}
payload = {
    "messages": [
        {"role": "user", "content": "请帮我写一段关于人工智能的摘要"}
    ],
    "model": "deepseek-chat"
}

response = requests.post(url, json=payload, headers=headers)
print(response.json())
const response = await fetch("https://nextool-api-proxy-vercel.vercel.app/api/chat", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "X-RapidAPI-Key": "your-rapidapi-key",
    "X-RapidAPI-Host": "nextool-api-proxy-vercel.vercel.app"
  },
  body: JSON.stringify({
    messages: [
      { role: "user", content: "请帮我写一段关于人工智能的摘要" }
    ],
    model: "deepseek-chat"
  })
});

const data = await response.json();
console.log(data);

响应示例

200 OK application/json 
{
  "choices": [
    {
      "message": {
        "content": "人工智能（AI）是计算机科学的一个分支...",
        "role": "assistant"
      }
    }
  ]
}

GET /api/usage/{ip} 查询IP使用量

查询指定IP地址的API调用次数和限额信息，用于监控使用情况。

路径参数

参数名类型必填说明
ip string 必填 要查询的IP地址，如 192.168.1.1

代码示例

curl https://nextool-api-proxy-vercel.vercel.app/api/usage/192.168.1.1 \
  -H "X-RapidAPI-Key: your-rapidapi-key" \
  -H "X-RapidAPI-Host: nextool-api-proxy-vercel.vercel.app"
import requests

ip = "192.168.1.1"
url = f"https://nextool-api-proxy-vercel.vercel.app/api/usage/{ip}"
headers = {
    "X-RapidAPI-Key": "your-rapidapi-key",
    "X-RapidAPI-Host": "nextool-api-proxy-vercel.vercel.app"
}

response = requests.get(url, headers=headers)
print(response.json())
const ip = "192.168.1.1";
const response = await fetch(
  `https://nextool-api-proxy-vercel.vercel.app/api/usage/${ip}`,
  {
    headers: {
      "X-RapidAPI-Key": "your-rapidapi-key",
      "X-RapidAPI-Host": "nextool-api-proxy-vercel.vercel.app"
    }
  }
);
const data = await response.json();
console.log(data);

响应示例

200 OK application/json 
{
  "ip": "192.168.1.1",
  "hourly_count": 5,
  "limit": 10
}
