2

创建数字产品

为每个 NexTool 产品创建对应的数字商品，设置价格和自动发货内容。

  • 进入「卖家中心」→「发布商品」
  • 商品类型选择「数字商品 / 虚拟商品」
  • 填写商品名称、描述、价格
  • 上传商品封面图（建议 800×600）
  • 设置自动发货内容（API Key / 激活码 / 使用教程链接）

3

获取支付链接

每个商品创建后会生成唯一的购买链接，将用于产品页面中的「立即购买」按钮。

  • 在卖家中心找到已创建的商品
  • 点击「分享」或「复制链接」
  • 链接格式通常为：https://mianbaoduo.com/o/xxxxx
  • 记录每个产品对应的支付链接
# 链接格式示例
ViralCopy: https://mianbaoduo.com/o/abc123
SmartSummary: https://mianbaoduo.com/o/def456
CodeMentor: https://mianbaoduo.com/o/ghi789
全工具套餐: https://mianbaoduo.com/o/jkl012

4

替换产品页面中的占位链接

将产品页面中的 #PLACEHOLDER_LINK# 替换为真实的支付链接。

  • 打开每个产品页面的 HTML 文件
  • 搜索 # 或 placeholder
  • 将占位链接替换为步骤3获取的真实链接
  • 确认所有「立即购买」按钮指向正确的支付链接
<!-- 替换前 -->
<a href="#PAYMENT_LINK_VIRALCOPY#">立即购买</a>

<!-- 替换后 -->
<a href="https://mianbaoduo.com/o/abc123">立即购买</a>
