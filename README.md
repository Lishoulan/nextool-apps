## 🚀 快速开始

### 本地开发

```bash
# 克隆项目
git clone https://github.com/Lishoulan/nextool-apps.git

# 进入项目目录
cd nextool-apps

# 使用任意静态服务器启动（例如 Python）
python -m http.server 8080

# 或使用 Node.js 的 serve
npx serve .
```

### 部署

项目使用 GitHub Actions 自动部署到 GitHub Pages，推送至 `main` 分支即可自动部署。
